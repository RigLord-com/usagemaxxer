# Security Policy

## Reporting a Vulnerability

Do not open a public issue for a suspected vulnerability, token exposure, or
credential-handling problem. Report it privately through the repository's
[Security Advisories](../../security/advisories/new) page.

Include the affected version, reproduction steps, and potential impact. Do not
include live credentials or tokens. You should receive an initial response
within seven days.

## Supported Versions

No public version has been released yet. Security fixes apply to the current
release candidate until a versioned support policy is published.
